# Project operation

### Tool and environment configuration

代码运行平台：Visual Studio 2019

界面设计平台：Qt

插件：Qt VS Tool 

环境配置：opencv

### Deployment

- 配置VS2019环境，安装Qt及Qt VS Tool。
- 添加Qxt插件到Qt中
- 创建Qt Widgets Application项目
- 将项目中得源代码文件及ui文件添加在Qt Widgets Application中

### Debug/Release

可以在VS中使用debug模式直接运行程序，也可以在release模式下生成exe文件。
